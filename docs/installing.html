安装¶

通过Python包¶

安装包（或将其添加到``requirements.txt``文件中）:

pip install sphinx_rtd_theme

在你的``conf.py``文件中:

html_theme = "sphinx_rtd_theme"

通过Git或下载¶

将``sphinx_rtd_theme / sphinx_rtd_theme``存储库符号链接或子树化到文档“docs / _themes / sphinx_rtd_theme`”中，然后将以下两个设置添加到Sphinx``conf.py``文件中：

html_theme = "sphinx_rtd_theme"
html_theme_path = ["_themes", ]
