阅读Docs Sphinx主题¶

sphinx_rtd_theme 是一个sphinx_主题, 旨在展现现代感和移动友好性.这个主题主要用于 readthedocs.org, 但可以与您自己的sphinx项目一起使用.要阅读更多内容并查看正在运行的演示文件, 请访问 readthedocs.org.

安装¶

主题在 PyPI 上分发, 可以用pip安装:

pip install sphinx_rtd_theme

有关更多信息, 请阅读完整的安装文档 here.

配置¶

sphinx_rtd_theme 在页面级别和全局级别上都是高度可定制的.要查看所有可能的配置选项, 请阅读配置文档 here.

特约¶

如果您想帮助改进主题或在叉子的情况下更好地控制主题, 请阅读我们的贡献指南 here.
